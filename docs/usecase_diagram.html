«system» Agri-Brain Platform
UC-01: Monitor Soil Health
Actor: ESP32, AMD Gateway
  • ESP32 reads moisture + pH
  • Publishes to MQTT telemetry
  • Soil Brain analyses per-grid
  • Alert if below threshold
UC-02: Auto Irrigation
Actor: AMD Gateway, ESP32
  • Moisture < 50% detected
  • Grid added to queue
  • Sequential valve opens
  • Timer auto-closes valve
UC-03: Manual Irrigation
Actor: Farmer, AMD Gateway
  • Farmer taps web/mobile
  • Selects region + duration
  • MQTT command dispatched
  • ESP32 opens valve
UC-04: Voice Command
Actor: Farmer, AMD Gateway
  • Farmer speaks in Kannada/Hindi
  • NLP Brain parses intent
  • Irrigation / logging action
  • Voice feedback returned
UC-05: Crop Disease Detection
Actor: Farmer, AMD Gateway
  • Farmer uploads leaf photo
  • Vision Brain analyses locally
  • Disease/pest type returned
  • Alert + treatment advice given
UC-06: Motor Guard
Actor: ESP32, AMD Gateway
  • ESP32 sends PSI reading
  • Acoustic Brain checks anomaly
  • Emergency shutdown triggered
  • All valves close instantly
UC-07: AI Farm Advisory
Actor: Farmer, Gemini API
  • Farmer asks advisory question
  • Gemini Bridge forwards query
  • AI response returned
  • Shown in dashboard chat
UC-08: Digital Ledger
Actor: Farmer, AMD Gateway
  • Voice / action captured
  • NLP extracts material + qty
  • Stored in SQLite DB
  • Dashboard shows history
UC-09: System Heartbeat
Actor: AMD Gateway
  • Gateway publishes heartbeat/5s
  • Dashboard shows ONLINE status
  • Disconnection alert raised
